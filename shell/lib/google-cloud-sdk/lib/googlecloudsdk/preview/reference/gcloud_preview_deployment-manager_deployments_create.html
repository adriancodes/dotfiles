NAME
gcloud preview deployment-manager deployments create - insert (Create) a deployment using a specified template
SYNOPSIS
gcloud preview deployment-manager deployments create   DEPLOYMENT   [--api-version   API_VERSION;   default="v1beta2"]   [--description   DESCRIPTION]   [--format   FORMAT]   [--help]   [--no-poll-for-completion]   [--overrides   [OVERRIDES   …]]   [--polling-period   POLLING_PERIOD;   default="10"]   [--project   PROJECT]   [--quiet,   -q]   --template   TEMPLATE   [-h]
DESCRIPTION
Insert (Create) a deployment using a specified template.
POSITIONAL ARGUMENTS
DEPLOYMENT
Deployment name.
FLAGS
--description DESCRIPTION
Deployment description.
--no-poll-for-completion
Do not poll continuously for deployment completion. If set, only one Get API call is made. (default=False)
--overrides [OVERRIDES …]
Optional "Path=Value Path=Value …" template overrides. Overrides must be a list of space delimited key=value pairs. This overrides existing template settings for this deployment but does not change the template itself. Example: --overrides $.modules.foo.replicaPoolModule.numReplicas=20 $.modules.bar.replicaPoolModule.numReplicas=5 …
--polling-period POLLING_PERIOD; default="10"
Number of seconds to sleep between consecutive polls. Only effective when --poll-for-completion is provided.
--template TEMPLATE
Name of the template to use. The template must exist in the system before creating the deployment.
GLOBAL FLAGS
--api-version API_VERSION; default="v1beta2"
Optional API version.
--format FORMAT
Specify a format for printed output. By default, a command-specific human-friendly output format is used. Setting this flag to one of the available options will serialize the result of the command in the chosen format and print it to stdout. Supported formats are: json, text, yaml.
--help
Display detailed help.
--project PROJECT
The Google Cloud Platform project name to use for this invocation. If omitted then the current project is assumed.
--quiet, -q
Disable all interactive prompts when running gcloud commands. If input is required, defaults will be used, or an error will be raised.
-h
Print a summary help and exit.
NOTES
This command is in the Google Cloud SDK preview component. See installing components if it is not installed.
