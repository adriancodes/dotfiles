NAME
gcloud preview - manage Preview CLI command groups
SYNOPSIS
gcloud preview   GROUP   |   COMMAND   [--format   FORMAT]   [--help]   [--project   PROJECT]   [--quiet,   -q]   [-h]
DESCRIPTION
Manage Preview CLI command groups.
FLAGS
--format FORMAT
Specify a format for printed output. By default, a command-specific human-friendly output format is used. Setting this flag to one of the available options will serialize the result of the command in the chosen format and print it to stdout. Supported formats are: json, text, yaml.
--help
Display detailed help.
--project PROJECT
The Google Cloud Platform project name to use for this invocation. If omitted then the current project is assumed.
--quiet, -q
Disable all interactive prompts when running gcloud commands. If input is required, defaults will be used, or an error will be raised.
-h
Print a summary help and exit.
GROUPS
GROUP is one of the following:
app
Manage your App Engine app.
autoscaler
Manage autoscalers of cloud resources.
container
Deploy and manage clusters of machines for running containers.
datastore
Manage your Cloud Datastore.
deployment-manager
Manage Deployments of cloud resources.
instance-groups
Manage Cloud Instance Groups.
managed-instance-groups
Managed Instance Groups client.
replica-pools
Manage cloud replica pools.
resource-views
Manage Cloud Resource Views.
