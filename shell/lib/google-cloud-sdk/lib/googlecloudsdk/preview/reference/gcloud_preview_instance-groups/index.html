NAME
gcloud preview instance-groups - manage Cloud Instance Groups
SYNOPSIS
gcloud preview instance-groups   GROUP   |   COMMAND   [--api-version   API_VERSION;   default="v1beta2"]   [--format   FORMAT]   [--help]   [--project   PROJECT]   [--quiet,   -q]   --zone   ZONE   [-h]
DESCRIPTION
Manage Cloud Instance Groups.
FLAGS
--api-version API_VERSION; default="v1beta2"
Optional API version.
--zone ZONE
Instance group zone name.
GLOBAL FLAGS
--format FORMAT
Specify a format for printed output. By default, a command-specific human-friendly output format is used. Setting this flag to one of the available options will serialize the result of the command in the chosen format and print it to stdout. Supported formats are: json, text, yaml.
--help
Display detailed help.
--project PROJECT
The Google Cloud Platform project name to use for this invocation. If omitted then the current project is assumed.
--quiet, -q
Disable all interactive prompts when running gcloud commands. If input is required, defaults will be used, or an error will be raised.
-h
Print a summary help and exit.
GROUPS
GROUP is one of the following:
instances
Manage instances in an instance-group.
COMMANDS
COMMAND is one of the following:
add-service
Adds a service to one or all instances in an instance group.
create
Insert (create) an instance group.
delete
Delete an instance group.
get
Gets information about a single instance group.
list
List all instance groups for a given project.
list-services
Lists the services for a resource or an instance group.
remove-service
Removes a service to one or all instances in an instance group.
NOTES
This command is in the Google Cloud SDK preview component. See installing components if it is not installed.
