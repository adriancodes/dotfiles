NAME
gcloud preview replica-pools - manage cloud replica pools
SYNOPSIS
gcloud preview replica-pools   GROUP   |   COMMAND   [--api-version   API_VERSION;   default="v1beta1"]   [--format   FORMAT]   [--help]   [--project   PROJECT]   [--quiet,   -q]   --zone   ZONE   [-h]
DESCRIPTION
Manage cloud replica pools.
FLAGS
--api-version API_VERSION; default="v1beta1"
Optional API version.
--zone ZONE
Replica pool zone name
GLOBAL FLAGS
--format FORMAT
Specify a format for printed output. By default, a command-specific human-friendly output format is used. Setting this flag to one of the available options will serialize the result of the command in the chosen format and print it to stdout. Supported formats are: json, text, yaml.
--help
Display detailed help.
--project PROJECT
The Google Cloud Platform project name to use for this invocation. If omitted then the current project is assumed.
--quiet, -q
Disable all interactive prompts when running gcloud commands. If input is required, defaults will be used, or an error will be raised.
-h
Print a summary help and exit.
GROUPS
GROUP is one of the following:
replicas
Manage replicas in a replica pool.
COMMANDS
COMMAND is one of the following:
create
Insert (Create) a replica pool.
delete
Deletes a replica pool and the corresponding replicas.
get
Gets information about a single replica pool.
list
List all replica pools in a given project.
resize
Resizes a replica pool to a provided size.
update-template
Updates the template for an existing replica pool.
NOTES
This command is in the Google Cloud SDK preview component. See installing components if it is not installed.
